The following tools are available:

- React
- Webpack
  - Bundle Analyzer
- Jest
- Babel
- ESLint
- Prettier
- Husky

## Scripts

Run Development Server with Hot Reloading

```
npm run dev:hot
```

Run Development Server without Hot Reloading

```
npm run dev
```

Run Storybook

```
npm run storybook
```

Build the project

```
npm run build
```

Run Unit Tests with Jest

```
npm run test
```

Format with Prettier

```
npm run format
```

Run ESLint

```
npm run lint
```
